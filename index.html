⚠️ Per funzionare al 100% su iPad

Questa app deve essere su GitHub Pages (gratis) per importare correttamente in Calendario iOS. Dal tuo iPad:

  • 1Vai su github.com con Safari → crea account gratuito
  • 2Crea nuovo repository → chiamalo turni-rai
  • 3Add file → Create new file → chiamalo index.html
  • 4Incolla tutto il codice di questa pagina → Commit
  • 5Settings → Pages → Branch: main → Save
  • 6Dopo 1 min: tuonome.github.io/turni-rai
  • 7Apri quell'URL in Safari → Condividi → Aggiungi alla schermata Home ✅
